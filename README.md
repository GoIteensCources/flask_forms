## Шаблон для бдуь якого доданку на Flask

### Необхідні бібліотеки:
-  Flask
-  Flask-SQLAlchemy
-  Flask-WTF

```sh 
pip install flask flask-sqlalchemy flask-wtf python-dotenv
```
